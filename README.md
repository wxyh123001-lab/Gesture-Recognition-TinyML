# -Embedded-Gesture-Recognition-TinyML-
This project builds upon an existing embedded machine learning system based on the Arduino Nano 33 BLE Sense and the Magic Wand example provided in the Harvard_TinyMLx package.

Key Features

• Keras-based implementation
• Five different vision architecture styles
• Unified training and evaluation pipeline
• TFLite conversion for deployment
• Benchmarking and visualization
• Easy to extend to custom datasets

───

Use Cases

This project can be used for:

• comparing deep learning architectures
• learning Keras Functional API design
• testing lightweight deployment workflows
• preparing image models for TFLite edge inference
• academic assignments and prototyping

───

Notes

• The notebook currently uses MNIST for demonstration.
• Some architectures are simplified educational versions of the original papers to make them easier to understand and run in notebook environments.
• For production deployment, further optimization and task-specific tuning may be needed.
